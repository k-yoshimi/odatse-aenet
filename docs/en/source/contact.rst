Bug Reports
-----------

Please report bugs via GitHub Issues:

https://github.com/issp-center-dev/ODAT-SE/issues

When filing a report, please include the following information:

- Versions of ODAT-SE and odatse-aenet
- OS and Python versions
- Input files (e.g., input.toml)
- Error messages or output
